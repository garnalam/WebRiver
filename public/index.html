Hướng Dẫn Sử Dụng

Các bước sử dụng:

  1. Chọn một ảnh (định dạng RGB).
  2. Chọn file CSV chứa metadata (xem file mẫu).
  3. Nhập tháng/năm của ảnh (mm/yy).
  4. Nhấn "Dự Đoán" để bắt đầu.
  5. Xem log, chọn ngày dự đoán từ lịch.
  6. Xem kết quả và tải CSV nếu cần.
